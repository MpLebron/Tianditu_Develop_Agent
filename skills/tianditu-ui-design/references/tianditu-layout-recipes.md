# Tianditu Layout Recipes

天地图智能体常见三类页面的布局配方。优先复用，不要每次重搭。

## A. 首页（Hero + 案例卡片）

适用：产品首页、能力介绍页、样例入口页。

- 顶部：品牌 + 主行动按钮（开始使用）
- 中部：一句主标题 + 一句说明 + 输入区（可选）
- 下部：案例卡片网格

建议尺寸：

- 页面最大内容宽度：`1200px`
- 卡片最小宽度：`280px`
- 卡片网格：`grid-template-columns: repeat(auto-fill, minmax(280px, 1fr))`
- 卡片内结构：标签 > 标题 > 描述 > 行动链接

## B. 工作区（左侧控制 + 右侧地图）

适用：POI 搜索、路径规划、专题分析。

- 左栏固定宽度：`320~380px`
- 右侧地图：剩余空间自适应
- 左栏内部结构：工具区（顶部固定）+ 结果区（滚动）

关键交互：

- 结果列表点击 -> 地图定位
- 地图点击 -> 左栏详情同步
- 滚动区域只保留一个主滚动容器，避免嵌套滚动

## C. 小屏布局（底部抽屉）

适用：移动端地图场景。

- 地图占全屏
- 控制面板改为底部抽屉（可展开/收起）
- 关键动作按钮固定在右下角

断点建议：

- `<= 1024px`：侧栏缩窄 + 字号略降
- `<= 768px`：切换到底部抽屉模式
