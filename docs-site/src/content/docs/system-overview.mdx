这套系统不是单纯的“地图页面生成器”。更准确地说，它是一套面向天地图项目的智能开发工作台：你可以提出需求、上传数据、获得可运行页面、继续修改、触发自动修复、回看运行档案，并最终生成可分享的成果页。

<Aside type="note" title="一句话理解">
你可以把它理解成“地图专题制作 + 数据上图 + 自动修复 + 分享管理”四件事合在一起的一套交付型系统。
</Aside>

## 它具体能做什么

  <Card title="自然语言生成地图页面" icon="sparkles">

## 最终会得到什么

1. 一个可运行、可继续修改的地图页面。
2. 一条可用于内部审阅或对外展示的分享链接。
3. 一份可回看生成、修复和巡检过程的运行记录。

## 典型的使用节奏

1. 先讲清楚主题和目标，不用第一次就把所有细节写满。
2. 先拿到第一版结构，再继续要求补风格、补交互、补联动。
3. 如果数据上图不对，先确认数据结构，不要直接跳到美化步骤。
4. 页面接近交付时，再重点看修复、巡检、分享与管理。
